# Sky-Hackathon-8th

该库用来存储参加 NVIDIA Sky-Hackathon 8th 产生的相关工程文件，
本次比赛围绕数据合成与目标检测两个模块展开，
首先使用 NVIDIA Omniverse™ 完成目标检测的训练数据的合成工作，然后使用 NVIDIA TAO 完成模型的训练工作
